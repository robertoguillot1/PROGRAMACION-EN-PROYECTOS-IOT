# 🌿 HydroIoT: Sistema de Riego Hidropónico Automatizado
### Proyecto de Electrónica en Proyectos IoT - Universidad de La Guajira (2026)

Este proyecto desarrolla un sistema de monitoreo y control de riego basado en el microcontrolador ESP32 para cultivos hidropónicos, específicamente diseñado para el contexto climático de Riohacha, La Guajira, Colombia.

## 🔧 Características Técnicas
*   **Hardware**: ESP32, Sensor DHT22 (Temperatura/Humedad), Sensor de Humedad de Suelo, Módulo Relay, LCD 20x4 I2C.
*   **Firmware**: Implementado en C++ (Arduino IDE) con servidor web embebido.
*   **Dashboard**: Interfaz de alta fidelidad con Three.js para visualización 3D y Telemetría IoT.
*   **IA Agro-Asistente**: Módulo de diagnóstico inteligente basado en el estado del cultivo.

## 🔬 Objetivos
1. **Analizar** condiciones agroambientales en Riohacha.
2. **Diseñar** arquitectura IoT para monitoreo y control.
3. **Implementar** firmware con sensores y servidor web.
4. **Evaluar** eficiencia hídrica y crecimiento vegetal.
